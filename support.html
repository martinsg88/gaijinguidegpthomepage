How to use Gaijin AI Guide

Using the Gaijin AI Guide is straightforward. In the future, we plan to expand our support to include various locations in Japan. However, currently, we only offer support for Tokyo. Please select a genre from the predefined list.

Input required fields

Using the system is straightforward. Simply fill in each of the required fields, with the last two being free text inputs. The fields "Locations," "Duration," and "Number of People" are selection pickers. Once you have entered all of the necessary information, press the "Let's find out" button. This will trigger a query to ChatGPT, which will return up to 10 recommendations for your selected area.

Input required fields cont.

You can view an example of the selection picker fields on this page. In order to enhance the prompt engineering process, some of the fields are pre-defined to maximize the capabilities of ChatGPT.

It's that simple!

After pressing the "Let's find out" button, you have the option to refine your query and try again or, if you are satisfied with the output, you can visit one of the recommended locations using the provided Google Maps link. We hope you have a great experience and enjoy using our tool!
